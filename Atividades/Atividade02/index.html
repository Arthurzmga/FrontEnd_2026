Funções da linguagem C

Abaixo serão listadas as funções junto de uma explicação breve do que ela faz.

Printf()

Quando iniciamos os estudos na linguagem C, é comum nós depararmos com a função printf(), pois ela é a função que fará uma mensagem ser exibida no console.

Console

printf("Hello world!");

Scanf()

A função scanf() é uma função utilizada para "escanear" o que o usuário digitar, armazenando-o em uma variável. Porém, para que possa guardar o que foi escrito é necessário alguns parâmetros para que funcione, como: "%d", "%f", "%c", e "%s". Além de que, é necessário indicar qual é a variável, sendo necessário colocar um '&' antes dela.

Console

scanf("%d", &var);

If-else()

Agora o if-else, ele se trata de uma função que tomará decisões baseadas em condições, por exemplo, se um número for maior que outro numero o sistema tomará certa ação. E existem diversos tipos de operadores lógicos, como: maior, menor, maior ou igual, e menor ou igual. Além de existir operadores booleanos, que permitem mais condições, como por exemplo, joão tem 18 anos e mora em maringá, logo ele pode beber em algum local na cidade. Mas, também pode ser utilizado o ou, por exemplo, para validarmos o sexo, onde deve ser digitado M(masculino) ou F(feminino), e se for digitado algo diferente, ele deve digitar novamente até que seja um dos dois.

Console

if(a > b) {
printf("Sim, a variável A é maior que a B!");
} else {
printf("Não, a variável A não é maior que a B!");
}

While()

A função while() é uma estrutura de repetição. Ela irá executar um bloco de código enquanto uma condição for verdadeira. Ou seja, enquanto a condição for atendida, o código continuará rodando. Ela é muito utilizada quando não sabemos exatamente quantas vezes o código deverá repetir.

Console

int contador = 0;

while(contador < 5) {
printf("O contador está em: %d\n", contador);
contador++;
}

No exemplo acima, o código será executado enquanto contador for menor que 5.

For()

A função for() também é uma estrutura de repetição, porém ela é mais utilizada quando já sabemos quantas vezes queremos que o código seja executado.
Ela possui três partes:

  • Inicialização
  • Condição
  • Incremento

      • Console

      for(int i = 0; i < 5; i++) {
      printf("Valor de i: %d\n", i);
      }

      Nesse caso, o código será executado 5 vezes.

Do-While()

O do-while() é muito parecido com o while(), porém existe uma diferença importante: ele executa pelo menos uma vez, mesmo que a condição seja falsa.

Console

int numero;

do {
printf("Digite um número maior que 10: ");
scanf("%d", &numero);
} while(numero <= 10);

Aqui, o usuário será obrigado a digitar até que informe um número maior que 10.

Switch()

O switch() é uma estrutura de seleção utilizada quando queremos comparar o valor de uma variável com múltiplas opções. Ele torna o código mais organizado quando temos várias condições baseadas no mesmo valor.

Console

int opcao;

printf("Escolha uma opção (1-3): ");
scanf("%d", &opcao);

switch(opcao) {
case 1:
printf("Você escolheu a opção 1.");
break;
case 2:
printf("Você escolheu a opção 2.");
break;
case 3:
printf("Você escolheu a opção 3.");
break;
default:
printf("Opção inválida.");
}

Nesse exemplo, o programa executa um bloco específico de código dependendo do valor digitado pelo usuário. O break é utilizado para impedir que os próximos casos sejam executados.
